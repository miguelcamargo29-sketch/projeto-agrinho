# Site - Consultor de Imóveis (M. Lopez)

Este projeto é uma versão estática da página inicial criada no Wix Harmony,
separada em:

- `index.html` – estrutura da página
- `style.css` – estilos
- `script.js` – espaço para scripts (atualmente simples)

## Como usar

1. Coloque os três arquivos na mesma pasta:
   - `index.html`
   - `style.css`
   - `script.js`
2. Abra o `index.html` no navegador.
